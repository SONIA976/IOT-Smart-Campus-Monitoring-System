# Hardware Components

The following hardware components are used in the IoT Smart Campus Monitoring System:

1. NodeMCU (ESP8266) – Microcontroller used for processing and IoT communication.
2. PIR Motion Sensor (HC-SR501) – Detects human movement.
3. Relay Module – Controls the ON/OFF state of the light bulb.
4. LED Bulb / Lamp – Represents the lighting system.
5. Breadboard – Used for temporary circuit connections.
6. Jumper Wires – Used to connect components.
7. Power Supply / USB Cable – Powers the NodeMCU.

## Description

The PIR motion sensor detects movement and sends a signal to the NodeMCU.  
The NodeMCU processes this signal and activates the relay module.  
The relay then controls the light bulb, turning it ON or OFF depending on motion detection.
