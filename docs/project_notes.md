## Project Description
The IoT Smart Campus Monitoring System is designed to automate campus lighting using motion detection. 
A PIR motion sensor detects human movement and sends the signal to the NodeMCU microcontroller. 
The NodeMCU processes the signal and activates a relay module to control the lighting system. 
This helps reduce unnecessary power consumption and improves energy efficiency.
