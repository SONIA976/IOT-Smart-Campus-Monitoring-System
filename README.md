# IoT Smart Campus Monitoring System

An IoT project that detects human movement using motion sensors and automatically controls lighting for energy-efficient light management.

## Project Overview
This project presents the design and implementation of an IoT-based Smart Campus Monitoring System with a smart lighting mechanism using motion sensors. The system automatically detects human movement and controls lighting to reduce energy consumption.

## Features
- Automatic light control using PIR motion sensor
- IoT-based monitoring using NodeMCU (ESP8266)
- Energy-efficient smart lighting system
- Real-time monitoring capability

## Hardware Components
- NodeMCU (ESP8266)
- PIR Motion Sensor (HC-SR501)
- 1 Channel Relay Module
- LED / Light Bulb
- Breadboard
- Jumper Wires
- Power Supply

## Software Used
- Arduino IDE
- Embedded C / Arduino Programming
- IoT Platform (Blynk / ThingSpeak)

## System Workflow
PIR Motion Sensor → NodeMCU → Relay Module → Light Control

## Objective
To develop an IoT-based smart lighting system that detects human presence and automatically controls campus lighting to improve energy efficiency.
