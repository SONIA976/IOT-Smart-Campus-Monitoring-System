# Block Diagram

PIR Motion Sensor → NodeMCU (ESP8266) → Relay Module → Light Bulb
                              ↓
                        IoT Platform

## Description
The PIR motion sensor detects human movement and sends the signal to the NodeMCU microcontroller. 
The NodeMCU processes the signal and activates the relay module, which controls the light bulb. 
The system can also send data to an IoT platform for monitoring.
